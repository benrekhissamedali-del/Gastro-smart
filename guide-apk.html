Projet prêt à uploader

Le fichier restogest-apk-projet.zip contient tout ce dont vous avez besoin.

Extrayez-le sur votre ordinateur avant de commencer.

1
3 minutes

Créez un compte GitHub

GitHub est une plateforme gratuite. C'est là que votre code va vivre et que le build va tourner.

🔗 github.com/signup →
  1. Entrez votre email et créez un mot de passe
  2. Choisissez un nom d'utilisateur (ex: mohamedbenali)
  3. Vérifiez votre email
  4. Choisissez le plan Free
2
2 minutes

Créez un dépôt (repository)

Un dépôt, c'est un dossier qui stocke votre code sur GitHub.

🔗 github.com/new →
  1. Repository name : restogest-mobile
  2. Cochez Private (pour garder votre code privé)
  3. Ne cochez PAS "Add README file"
  4. Cliquez sur Create repository
3
5 minutes

Uploadez les fichiers

Extrayez le ZIP et uploadez tout son contenu sur GitHub.

  1. Sur la page de votre nouveau dépôt, cliquez sur "uploading an existing file" (lien bleu)
  2. Glissez-déposez TOUS les fichiers extraits du ZIP dans la zone
  3. ⚠️ Important : incluez bien les dossiers .github et le fichier .gitignore (qui commencent par un point)
  4. En bas, cliquez sur le bouton vert Commit changes
Si vous ne voyez pas les fichiers cachés (.github, .gitignore) :
Windows : Explorateur → Affichage → Éléments masqués
Mac : Appuyez sur Cmd + Shift + . dans le Finder
4
1 clic

Lancez le build

GitHub va compiler votre APK dans le cloud, automatiquement.

  1. Cliquez sur l'onglet Actions en haut de la page du dépôt
  2. Si demandé, cliquez sur le gros bouton vert "I understand my workflows, go ahead and enable them"
  3. Le build démarre tout seul ! 🎉
Si le build ne démarre pas automatiquement : cliquez sur "Build Android APK" à gauche, puis "Run workflow" à droite.
5
8 à 12 minutes

Attendez la compilation

GitHub fait tout le travail à votre place.

  1. Un job apparaît avec un point jaune 🟡 (en cours)
  2. Cliquez dessus pour voir l'avancement en direct
  3. Les étapes deviennent vertes ✅ une par une
  4. Quand tout est vert, c'est prêt !
Profitez-en pour prendre un thé à la menthe. ☕🇹🇳
6
1 minute

Téléchargez l'APK

Votre fichier APK est prêt dans la section "Artifacts".

  1. Cliquez sur le job terminé (vert ✅)
  2. Scrollez jusqu'en bas de la page
  3. Dans "Artifacts", cliquez sur "RestoGest-APK"
  4. Un fichier ZIP se télécharge
  5. Extrayez-le → vous avez RestoGest-v1.0.0.apk 🎉
7
2 minutes

Installez sur votre téléphone

Transférez l'APK sur votre Android et installez-le.

  1. Envoyez l'APK à votre téléphone par WhatsApp, email, ou câble USB
  2. Sur le téléphone, ouvrez le fichier RestoGest-v1.0.0.apk
  3. Si Android bloque l'installation : Paramètres → Autoriser cette source
  4. Appuyez sur Installer
  5. Ouvrez RestoGest depuis votre liste d'apps 🍴
